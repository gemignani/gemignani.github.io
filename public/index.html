Brazilian Interactive DTV Applications

Brazilian DTV standard for interactive applications “Ginga” is now being followed by other South America countries. The standard is usually implemented by big electronics companies and setup-box manufacturers as a middleware service inside their platforms. Applications can be built using a special version of Java TV or NCL (Nested Context Language), the specification covers a wide range of concepts, so the developer should read the ABNT NBR standard before further adventures.
